# CapyBoard — Claude 開發規則

## 設計風格（重要：所有 UI 一律遵循）
CapyBoard 的視覺取向是**家庭、溫馨、輕鬆**。任何新增或修改 UI 一律採以下暖色系蠟筆調，**禁止使用冷色藍/灰作為主色或強調色**（藍灰僅用於系統性的錯誤/警示）。

### 色票
| 用途 | 色碼 | 備註 |
|------|------|------|
| 頁面背景 | `#FBF6EC` | 奶油底 |
| 卡片/面板背景 | `#FFFDF8` | 暖白 |
| 邊框 | `#F0E6D6` | 淺奶茶 |
| 主文字 | `#5C4A33` | 暖棕 |
| 次要文字 | `#A1907A` | 灰棕 |
| 主強調色（蜂蜜橘） | `#F4B860` / 深 `#E09B3D` | 主按鈕、選取態、active tab |
| 蜜桃珊瑚 | `#EF8E72` | 卡片類元件 |
| 抹茶綠 | `#8FBF9F` | 骰子類元件 |
| 薰衣草紫 | `#B89FD4` | 自訂類元件 |
| 提示/刪除（柔紅） | `rose-400` | 不用大紅，用柔和玫瑰色 |

### 造型準則
- **簡約**：少線條、少陰影層級；用柔和大圓角（`rounded-xl`～`rounded-2xl`）取代直角硬框
- **溫潤**：陰影用暖色低透明（如 `rgba(120,80,30,0.12)`），不用純黑陰影
- **元件造型**：遊戲元件（token）一律用「賭場籌碼」造型（外圈交錯邊點＋內圈虛線環＋中心 emoji），依類型套上方色盤，見 `GameEditor.tsx` 的 `TokenChip`
- **active 狀態**：用蜂蜜橘填底或描邊＋輕微放大/上浮，不用藍底
- **互動回饋**：hover 輕微上浮（`-translate-y-0.5`）、過渡 `transition-all`
